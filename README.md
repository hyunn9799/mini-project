# Yacht Dice Game
<img width="360" height="360" alt="image" src="https://github.com/user-attachments/assets/ffc106af-3941-4e85-920a-f2ec2abd387c" />


Yacht Dice Game은 5개의 주사위를 사용해 12턴 동안 다양한 점수 규칙을 채워가는 **클래식 주사위 게임**입니다. 전략적으로 규칙을 선택해 **최고 점수**를 노려보세요!

## 게임 목표
- **5개의 주사위**를 굴려 **총 12턴** 동안 **12가지 점수 규칙**에 점수를 기록합니다.
- 모든 턴 종료 후, **12개 규칙의 점수 총합**이 높은 사람이 승리합니다!

## 게임 규칙 상세

### 게임 진행
|12개 점수 항목이 "미사용"(-1) 상태로 초기화 
| 총 12턴 진행 
├─ 주사위 5개 굴리기 (1~6 난수) 
├─ 사용 가능 규칙 + 예상 점수 표시 
├─ 사용자 선택 → 점수 기록 
└─ 점수판 업데이트 
12턴 종료 → 최종 총점 계산 


### 12가지 점수 규칙

| **구분** | **규칙명** | **설명** | **점수 계산** |
|----------|------------|----------|---------------|
| **숫자 (6개)** | Ones ~ Sixes | 해당 숫자 주사위 합계 | `[2,2,3,5,5]` → Twos=4, Fives=10 |
| **조합 (6개)** | Choice | 주사위 5개 총합 | 항상 가능 |
| | Three of a Kind | 3개 이상 동일 → 총합 | 아니면 0점 |
| | Four of a Kind | 4개 이상 동일 → 총합 | 아니면 0점 |
| | **Full House** | 3개+2개 → **25점** 고정 | `[6,6,6,1,1]` |
| | **Small Straight** | 4연속 → **30점** 고정 | `1-2-3-4`, `2-3-4-5`, `3-4-5-6` |
| | **Large Straight** | 5연속 → **40점** 고정 | `1-2-3-4-5`, `2-3-4-5-6` |
| | **Yacht** | 5개 동일 → **50점** 고정 | `[4,4,4,4,4]` |
